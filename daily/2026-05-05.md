## What was built

### 1. Project restructure — monolith → multi-file
Split `office-planner.html` (1.3 MB, single file) into a maintainable multi-file project structure. A one-time Node.js extraction script (`scripts/extract-assets.js`) was written to perform the split.

**Output structure:**
```
index.html              ← clean HTML shell
css/styles.css          ← all styles
js/data/items.js        ← 8 product configurations
js/utils.js             ← fmtZAR, fmtSqm, escapeHtml helpers
js/pdf.js               ← jsPDF-based PDF generation
js/quote.js             ← webhook POST, form payload, quote view
js/main.js              ← catalogue render, stepper, form validation
assets/images/          ← 8 product JPEGs (decoded from base64)
vendor/jspdf.min.js     ← jsPDF 2.5.1 vendored
scripts/extract-assets.js
```

**Code quality fixes applied during review:**
- Stripped 2-space indent inherited from original `<script>` block context across all JS files
- Renamed `escape()` → `escapeHtml()` to avoid shadowing the deprecated browser global
- Added `escapeHtml()` to `it.desc` in the quote view for consistency
