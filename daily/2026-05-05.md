Split `office-planner.html` (1.3 MB, single file) into a maintainable multi-file project structure. A one-time Node.js extraction script (`scripts/extract-assets.js`) was written to perform the split.

**Output structure:**

### 7. Form security hardening

Security review of the quote form. XSS was already well-handled via `escapeHtml()`. Three additional fixes applied:

- **Removed PII console.log** (`js/quote.js`) — the full payload (name, email, phone) was being logged to the browser console, readable by any browser extension
- **Input `maxlength` attributes** (`index.html`) — caps payload size at the browser level before anything reaches the webhook:
  - company: 200 · contact: 100 · phone: 30 · email: 254 · location: 300
- **Honeypot anti-spam** (`index.html` + `js/main.js`) — hidden off-screen field (`#hp-website`) that real users never see or touch; bots fill it in automatically, submission is silently dropped
